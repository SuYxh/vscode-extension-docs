## 原生代码片段

插件已经内置了很多原生JavaScript代码片段,比如：

- clo ==> console.log(`object`, object)

- dob ==> const {propName} = objectToDescruct

- imp ==> import moduleName from 'module'
- imd ==> import { destructuredModule } from 'module'
- exp ==> export default moduleName
- exd ==> export { destructuredModule } from 'module'
- end ==> export const functionName = (params) => { }



文档： https://dashen.zhuanspirit.com/x/RoqrDQ

## 高级用法

代码片段支持可配置，一般会配置功能性的代码片段，可以在插件后台进行配置，配置成功后重启vscode即可！

后台地址：



### 配置方式

#### 新增代码片段

在代码片段菜单下新增代码片段即可！

- name：代码片段名称，一般与prefix保持一致即可。
- prefix：代码片段触发前缀
- body：代码体
- description：代码片段的描述
