2、可自定义代码片段的api，只需遵循如下接口格式即可！

```js
{
  "code": 200,
  "data": [
      {
        "prefix": "byteSize",   // 代码片段触发前缀
        "body": "const byteSize = str => new Blob([str]).size;",   // 代码体
        "description": "返回字符串的字节长度。"    // 代码片段描述
      }
  ]
}
```
